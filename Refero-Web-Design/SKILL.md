# refero-web-design

This is a public-safe teaching wrapper around `refero-design`.

## Role

Use it when the deliverable is specifically a website or web interface:

- landing page
- SaaS homepage
- pricing page
- product marketing page
- dashboard / web UI
- redesign / visual polish

## Workflow

1. Start from the user brief.
2. Use `refero-design` methodology to research styles, screens, and flows when available.
3. Synthesize a web direction: layout, section rhythm, typography, colors, CTAs, media strategy.
4. Convert the direction into a `DESIGN.md` or implementation-ready decision ledger.
5. Build or hand off to a coding workflow.
6. Audit with `taste-skill` / `hallmark`.

## Teaching line

`refero-design` is the design brain. `refero-web-design` is the web-delivery mode.
