# Prompt Patterns Library (/po)

## 1) Strategy Prompt

```text
Ты — стратег-консультант.
Цель: [цель]
Контекст: [контекст]
Собери 3 стратегии (агрессивная / сбалансированная / консервативная),
для каждой: шаги, риски, сроки, expected outcome.
```

## 2) Critic Prompt

```text
Ты — жёсткий ревьюер.
Проведи разбор плана: [план/идея]
Найди слабые места, скрытые риски, ложные допущения,
и предложи улучшенную версию.
```

## 3) Execution Prompt

```text
Ты — execution-менеджер.
Преобразуй цель [цель] в план на 7 дней.
Формат: День -> задачи -> метрика -> критерий завершения.
```

## 4) Content Prompt

```text
Ты — редактор-конвертер.
Нужно: [тип контента]
Аудитория: [аудитория]
Тон: [тон]
Сделай 3 варианта: короткий, средний, подробный.
```

## 5) Technical Prompt

```text
Ты — senior engineer.
Задача: [задача]
Стек: [стек]
Нужны: архитектура, пошаговая реализация, edge cases, тест-план.
```
